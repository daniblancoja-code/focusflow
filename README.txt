FOCUSFLOW - APP WEB SIMPLE

Qué incluye:
- Tareas
- Goals
- Hábitos
- Guardado local en el navegador
- Diseño móvil
- PWA básica (puedes añadirla a la pantalla de inicio)

Cómo usarla en tu ordenador:
1. Abre la carpeta productivity_app.
2. Haz doble clic en index.html.
3. La app se abrirá en tu navegador.

Cómo usarla en tu iPhone:
Opción sencilla:
1. Súbela a GitHub Pages, Netlify o Vercel.
2. Abre el enlace en Safari.
3. Pulsa Compartir > Añadir a pantalla de inicio.

Notas:
- Los datos se guardan localmente en el navegador.
- Si borras datos del navegador, se puede perder la información.
- Es una base MVP para seguir mejorándola después.
